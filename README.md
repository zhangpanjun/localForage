# localForage
localForage是一个基于indexedDB封装,类似于localStorage的缓存存取api的第三方库.实现indexedDB/webSql/localStorage不同方式存储，优雅降级，兼容性更高
# 使用模式如下：
1.在html中引入localForage.min.js
2.在下一个<script></script>中即可直接通过localforage.具体api()即可直接访问
